About the Project

This project focuses on detecting morphed facial images used in identity fraud. Morphing combines two or more faces to create a synthetic identity that can bypass traditional face recognition systems.

Objective

  • Detect facial morphing attempts
  • Improve biometric security
  • Prevent identity fraud

Project Modules

  • Face Data Collection
  • Face Recognition
  • Morph Detection
